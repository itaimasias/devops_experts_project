# Flask Hello World (Dockerized)

This is a simple Flask application that returns `Hello World` at the root URL.

## Prerequisites

- [Docker](https://www.docker.com/get-started) installed
- (Optional) [Docker Compose](https://docs.docker.com/compose/) installed
